# 算法知识图谱复习系统 V1 开发 TODO

## 1. 使用说明

1. 按阶段从上到下执行，优先完成 `P0`。
2. 每项完成后勾选 `[x]`，并在 PR/提交信息中引用任务编号（如 `T012`）。
3. 每个阶段结束前必须通过对应的验收项，不跳步。
4. 默认所有实现以 [PLAN.md](/Users/didi/work/knowledge_graph/PLAN.md) 为准。

## 2. 里程碑计划（3-4周）

1. Week 1：工程骨架 + 鉴权 + 数据模型 + 基础 API。
2. Week 2：知识点/关系/题目管理 + 复习引擎。
3. Week 3：随机训练 + 看板 + 导入导出。
4. Week 4：移动端优化 + 测试完善 + 部署上线。
